# Dashboard UI

UI dashboard с календарём, списком задач и таймером (в разработке).

## Stack
- HTML
- CSS
- JavaScript (ES Modules)
- Vite

## Features
- Календарь с переключением месяцев
- Выделение текущей даты
- Макет панели задач (UI)
- Базовая структура под таймер и уведомления

## Run locally

```bash
npm install
npm run dev
